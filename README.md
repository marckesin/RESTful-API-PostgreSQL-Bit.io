# RESTful API - PostgreSQL/Bit.io

API simples seguindo o padrão REST desenvolvida em Node/Express. Integração de banco de dados PostgreSQL usando DBaaS [Bit.io](https://www.bit.io).

## Endpoints

- GET - /users
- GET by id - /users/id
- POST - /users
- PUT - /users/id
- DELETE - /users/id
